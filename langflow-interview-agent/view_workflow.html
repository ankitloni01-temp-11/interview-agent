Why this design?

  • Dynamic Delegation: The Supervisor decides when to use an agent, rather than following a fixed script.
  • Central Intelligence: The Orchestrator layer keeps the "bigger picture," ensuring consistent responses.
  • Native LangGraph: This uses the official SupervisorPattern, moving all logic into the Graph Edges.
